# 📊 Planilha Gestão de Reservas de Salas — Clínica Escola de Psicologia

Projeto de reestruturação e automação de uma planilha Excel para gestão de reservas de salas, presença de atendimentos e acompanhamento operacional em uma Clínica Escola de Psicologia.

O objetivo foi transformar uma planilha operacional manual em uma solução mais segura, intuitiva e analítica, permitindo uso colaborativo por múltiplos usuários.

---

## 🚀 Objetivo do Projeto

Melhorar a gestão de agendamentos e presença em salas de atendimento por meio de:

- Padronização dos registros
- Redução de erros de preenchimento
- Proteção contra alterações indevidas
- Visualização gerencial com dashboard
- Interface mais amigável para usuários administrativos

---

## 🛠 Tecnologias Utilizadas

- Microsoft Excel

## 📂 Estrutura do Arquivo

### Aba `Agenda`
Contém os registros operacionais:

- Dia da semana
- Data
- Horário
- Sala
- Status do atendimento

### Aba `Legenda`
Tabela padronizada com categorias:

| Código | Significado |
|---|---|
| Atendimento | Sessão com paciente |
| Teste | Avaliação aplicada |
| Reunião | Encontro de equipe |
| Estudo | Atividade acadêmica |
| Supervisão | Supervisão clínica |
| Compareceu | Presença confirmada |
| Faltou | Ausência registrada |

### Aba `Dashboard`
Painel executivo com:

- Total de atendimentos
- Total de faltas
- Taxa de comparecimento
- Ocupação por sala
- Indicadores mensais
- Visual em estilo clínica (cores suaves)
